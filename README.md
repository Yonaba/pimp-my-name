Pimp My Name (r4)
============

![screenshot](screenshot.png)

Small old homebrew Lua application written in Lua for PSP

##What's this ?
Pimps out images and names using fonts.

##Notes
Compatible with PSP Fat and Slim (uses *LuaPlayerHm v6.6*)

##Installation Path
ms0:/PSP/GAME/
